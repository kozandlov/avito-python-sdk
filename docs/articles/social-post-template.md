# Шаблон анонса релиза

Новый релиз `avito-python-sdk` уже в PyPI.

Что внутри:
- typed response models для Avito API;
- async-only клиент `AvitoAsyncClient`;
- автоперегенерация SDK из OpenAPI;
- автоматическая публикация в PyPI.

Установка:

```bash
pip install avito-python-sdk
```

Репозиторий: https://github.com/kozandlov/avito-python-sdk  
PyPI: https://pypi.org/project/avito-python-sdk/
