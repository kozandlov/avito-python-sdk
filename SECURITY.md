# Security Policy

## Supported Versions

Поддерживаются последние релизные версии ветки `main` и последний опубликованный релиз в PyPI.

## Reporting a Vulnerability

Если вы нашли уязвимость:

1. Не создавайте публичный issue с деталями эксплуатации.
2. Отправьте описание через приватный канал владельцу репозитория.
3. Укажите:
   - шаги воспроизведения,
   - затронутую версию,
   - потенциальный impact.

После подтверждения проблемы будет подготовлен фикс и опубликован релиз.
